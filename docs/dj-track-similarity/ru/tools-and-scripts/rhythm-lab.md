# Rhythm Lab

> Audience: Пользователи этой страницы.
> Type: how-to

Common commands: `rhythm_lab_cli.py serve`, `train --profile <classifier-key> --source <library-db> --labels tools\rhythm-lab\data\rhythm_lab.sqlite`, and `promote --profile <classifier-key> --labels ...`. Use `--calibrate` and `promote --require-calibration` only deliberately.
