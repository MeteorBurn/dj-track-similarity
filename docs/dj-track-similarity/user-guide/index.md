# User Guide

Audience: day-to-day UI users  
Goal: use the browser app without reading internals first  
Type: how-to map

The UI is built around two working areas: the library on the left and search/set
workflows on the right. Use this section when you already have a scanned
database and want to work with tracks.

| Page | Use it for |
| --- | --- |
| [Browse the library](browse-library.md) | search, page, like, preview, seed, and inspect tracks |
| [Analyze the library](analyze-library.md) | run SONARA, MAEST, MERT, CLAP, and classifier jobs |
| [Search with seeds](search-with-seeds.md) | find related tracks through SONARA or MERT |
| [Smart Set Builder](smart-set-builder.md) | generate ordered read-only set previews |
| [Text search](text-search.md) | find music with CLAP prompts |
| [CLASS tab](class-tab.md) | use promoted classifier scores |
| [Export playlists](export-playlists.md) | write M3U or CSV set exports |
| [Tags and audio writes](tags-and-audio-writes.md) | understand the explicit genre tag-writing path |
