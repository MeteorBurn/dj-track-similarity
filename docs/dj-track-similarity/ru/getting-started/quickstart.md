# Быстрый старт: scan, analyze, search

> Audience: Новые пользователи, которым удобно работать в терминале.
> Goal: Создать локальную SQLite-базу, проверить зависимости и открыть UI с первыми результатами.
> Type: how-to

Этот маршрут создаёт локальную базу, проверяет FFmpeg и ML-зависимости, анализирует небольшую первую пачку и запускает браузерный UI. Начинайте с малого лимита: так проще заметить проблемы с декодером или пакетами до долгого анализа всей коллекции.

## 1. Установить и проверить

```powershell
python -m pip install -e ".[sonara,ml,rhythm-lab,dev]"
dj-sim doctor
```

Первая команда ставит приложение с дополнительными пакетами для анализа и Rhythm Lab. `dj-sim doctor` проверяет, что CLI запускается и видит нужные runtime-зависимости.

Готово, когда `dj-sim doctor` не сообщает критичных проблем. Если не найден FFmpeg, добавьте его в `PATH` или задайте `DJ_TRACK_SIMILARITY_FFMPEG`. Если падают импорты PyTorch, Torchaudio или TorchCodec, вернитесь к странице установки и проверьте согласованные версии пакетов.

## 2. Просканировать библиотеку

```powershell
dj-sim scan <music-folder> --db .\data\library.sqlite
```

`scan` читает теги и сохраняет пути плюс метаданные в SQLite. Аудиофайлы не переписываются.

Готово, когда команда выводит счётчики added, updated, unchanged и skipped, а файл базы создан по пути из `--db`.

## 3. Проанализировать первую пачку

```powershell
dj-sim analyze --models sonara,maest,mert,clap --limit 25 --db .\data\library.sqlite
```

Эта команда создаёт достаточно SONARA features и MAEST/MERT/CLAP результатов, чтобы попробовать вкладки поиска. Для всей библиотеки в CLI не указывайте `--limit`; в UI `Analyze limit = 0` означает всю библиотеку.

Готово, когда задача завершается без ошибок и анализированные треки появляются в результатах поиска. При проблемах декодирования добавьте `--diagnostics`.

## 4. Открыть UI

```powershell
dj-sim serve --host 127.0.0.1 --port 8765 --db .\data\library.sqlite
```

Откройте `http://127.0.0.1:8765/`, проверьте счётчики библиотеки, выберите seed tracks и попробуйте SET, SONARA, MERT или CLAP search.
