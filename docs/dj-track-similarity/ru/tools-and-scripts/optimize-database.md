# Оптимизация базы SQLite

> Для кого: Для тех, кто обслуживает локальное состояние SQLite.
> Задача: Запустить обслуживание с резервными копиями и проверками целостности.
> Тип: Руководство

```powershell
python scripts\optimize_database.py --db .\data\library.sqlite
```

Для фонотеки v7 скрипт работает с Core и обязательной соседней базой `*.artifacts.sqlite`. Он
проверяет их схемы и общий `catalog_uuid`, проверяет целостность, создаёт проверенные резервные копии
и затем выполняет обслуживание SQLite для каждого выбранного файла. В отчёте указаны пути, пути
копий, результаты проверки и размер до/после.

`*.evaluation.sqlite` хранит необязательные данные оценки и включается только когда существует и
предусмотрен текущей политикой скрипта. Скрипт не принимает структуру v5/v6, `*.timeline.sqlite` или
`*.representations.sqlite` как замену.

По возможности остановите основное приложение. Храните Core и Artifacts вместе для резервного
копирования и восстановления; обслуживание и копирование не изменяют исходное аудио.
