# Browse Library

> Audience: Пользователи этой страницы.
> Type: how-to

Library view uses server-side pagination. Search by artist, title, genre or path. Metadata dialog separates Mutagen tags, SONARA features, MAEST genres and classifier scores. Browser preview streams `/media/{track_id}`; AIFF can be transcoded to WAV for playback, but source audio is not rewritten.
