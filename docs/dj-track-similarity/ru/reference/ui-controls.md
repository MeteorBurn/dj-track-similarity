# Статус элементов UI

> Для кого: Для тех, кто ищет диапазоны настроек браузерного интерфейса.
> Задача: Не дать управлять библиотекой v7 через устаревшие элементы frontend.
> Тип: Справочник

Порт frontend на v7 отложен. Текущие элементы браузерного интерфейса не совместимы с v7, поэтому
здесь намеренно не заявляются актуальные флажки, диапазоны ползунков, размеры пакетов, действия
сброса или поведение выбора базы.

Для работы со схемой v7 используйте текущие контракты CLI и API. Допустимые имена результатов SONARA:
`core`, `timeline`, `embedding` и `fingerprint`; результата `Representations` и отдельного управления
базами Timeline/Representations нет. Core и обязательный `*.artifacts.sqlite` связаны через
`catalog_uuid`; `*.evaluation.sqlite` хранит необязательные данные оценки.

Команды анализа и границы хранения описаны в разделе [Анализ фонотеки в v7](../user-guide/analyze-library.md).
Безопасность исходных файлов описана в разделе [Локальная работа и безопасность](../concepts/local-first-safety.md).

Когда будет готов порт frontend v7, этот справочник нужно собрать заново по `frontend/src/api.ts` и
активным backend-схемам, а не переносить значения из удалённого UI v5/v6.
