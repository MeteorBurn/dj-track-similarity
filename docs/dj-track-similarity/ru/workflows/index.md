# Сценарии

> Для кого: Для тех, кто решает задачу с диджейской фонотекой, а не изучает отдельные функции.
> Задача: Дать практические рецепты и указать нужные страницы интерфейса и инструментов.
> Тип: Оглавление сценариев

Используйте этот раздел, когда знаете желаемый результат, но ещё не понимаете, какая модель или экран
поможет его получить.

## Выберите результат

| Результат | Сценарий |
| --- | --- |
| Возможный порядок вокруг нескольких опорных точек | [Подготовка сета](./prepare-a-set.md) |
| Короткий список вокруг одного референсного трека | [Поиск совместимых треков](./find-compatible-tracks.md) |
| Более широкий пул для последующего прослушивания | [Создание подборок](./build-crates.md) |
| Повторно используемая оценка личного признака | [Обучение персонального классификатора](./train-personal-classifier.md) |
| Безопасная процедура поддержания каталога в актуальном состоянии | [Обслуживание фонотеки](./maintain-library.md) |

Любой результат всё ещё требует проверки. Приложение сокращает и упорядочивает коллекцию, а
прослушивание решает, что останется.

## Страницы раздела

- [Подготовка сета](./prepare-a-set.md): референсные треки (seed), поиск, предварительный список SET, проверка Hybrid и экспорт.
- [Поиск совместимых треков](./find-compatible-tracks.md): путь от одного референсного трека к списку кандидатов.
- [Создание подборок](./build-crates.md): пулы для прослушивания и их экспорт.
- [Обучение персонального классификатора](./train-personal-classifier.md): превращение меток Rhythm Lab в опубликованную оценку.
- [Повторный анализ SONARA с раздельным хранением](./reanalyze-sonara-split-storage.md): миграция схемы v5 и пересоздание актуального анализа.
- [Обслуживание фонотеки](./maintain-library.md): обновление метаданных, отчёты, поиск дубликатов и безопасное исправление.

## Полезная привычка

Сначала работайте с предварительными списками. Экспортируйте или применяйте изменения только после прослушивания
или проверки отчёта.
