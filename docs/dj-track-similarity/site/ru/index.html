Локальный рабочий процесс

Сканируйте музыкальную папку, создавайте SQLite-библиотеку и запускайте анализ без загрузки файлов наружу.

🔬

Анализ несколькими моделями

Ритмические признаки SONARA, жанры MAEST, embeddings MERT и CLAP — запускайте только то, что нужно вашему workflow.

🔍

Гибкий поиск

Ищите похожие треки по seed-аудио через SONARA/MERT или описывайте звук обычным текстом через CLAP text-to-audio.

🤖

Пользовательские классификаторы

Обучайте promoted binary или multiclass classifier profiles по собственным размеченным трекам в Rhythm Lab.

🛠️

Служебные скрипты

Стабильные помощники для восстановления аудиометаданных, отчётов о дублях и оптимизации базы данных.

🔒

Безопасно для аудио по умолчанию

Сканирование, анализ, поиск и экспорт не изменяют аудиофайлы. Запись жанров — отдельное явное действие.
