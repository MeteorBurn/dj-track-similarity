# Как Smart Set Builder строит маршрут

> Для кого: Для тех, кто настраивает SET.
> Задача: Объяснить требования к анализу и причины порядка в черновике.
> Тип: Понятие

Smart Set Builder начинает с подходящих треков. Опорные точки и оценки кандидатов поступают в
построитель маршрута. Результат — черновик для прослушивания, а не автоматически готовый сет.

## Условия участия

Треку нужны данные SONARA, MERT, MAEST и CLAP. Ответ сообщает общее количество и отсутствующее
покрытие, чтобы было понятно, готова ли библиотека.

## Ручные и автоматические ориентиры

В ручном режиме выбранные опорные треки становятся фиксированными путевыми точками. В автоматическом
режиме первая точка выбирается из библиотеки с полным набором признаков, а затем подбираются
связанные ориентиры разных исполнителей.

Бэкенд ограничивает автоматические ориентиры диапазоном `1..5` и требует достаточное число
подходящих треков.

## Оценка кандидатов

Базовая оценка сочетает:

- сходство эмбеддингов MERT;
- сходство аудиоэмбеддингов CLAP;
- сходство эмбеддингов MAEST;
- широкое сходство SONARA;
- необязательное предпочтение и уверенность классификатора.

Режимы смещают баланс между близким сходством, открытием нового, странными соседними совпадениями и
плавным развитием.

## Упорядочивание

При построении последовательности добавляются:

- уверенность перехода;
- соответствие кривой энергии;
- соответствие кривой BPM, если она включена;
- давление разнообразия;
- развитие классификатора;
- давление повторов исполнителя.

Для известных исполнителей действует ограничение: не более одного трека в черновике. Если ручные
опорные треки нарушают правило, SET сообщает об ошибке, а не скрывает конфликт.

## Поведение BPM

Режим General BPM использует темп для совместимости перехода. Low-to-high и high-to-low добавляют
реальную траекторию BPM. Сопоставление половинного и двойного темпа помогает оценить переход, но не
заменяет фактическое движение.

SET сначала разрешает актуальные данные темпа SONARA. При уверенности ниже `0.45` рабочую оценку
могут подтвердить или заменить ранжированные кандидаты SONARA и тег BPM из Mutagen. Стабильность
битовой сетки ослабляет ненадёжные данные, а низкая надёжность сама по себе не может стать жёстким
исключением. См. [контракт SONARA v0.2.9](../reference/sonara-v0-2-9-contract.md).

## Добавление всегда явное

SET возвращает строки черновика и не заменяет текущий сет. Нажмите **Add preview**, чтобы добавить
треки.
