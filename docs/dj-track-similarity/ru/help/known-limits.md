# Известные ограничения

> Для кого: Для тех, кто проверяет, ожидаемо ли текущее поведение.
> Задача: Перечислить границы, не выдавая их за ошибки.
> Тип: Помощь

## Локальная работа зависит от локальной среды

Приложению нужны ваша среда Python, FFmpeg, необязательные ML-пакеты, среда GPU, пути файловой
системы и доступ браузера к бэкенду.

## Анализ моделей может быть тяжёлым

MAEST, MERT, MuQ и CLAP требуют дополнительных ML-зависимостей. Анализ большой фонотеки требует
времени и памяти. MuQ работает с аудио `float32` 24 кГц и сейчас используется только в LAB Reference
Compare, не в SET, Hybrid, Audio Dedup или при расчёте оценок классификаторов. Сначала задайте малый
`--limit`, затем настройте устройство и размеры пакетов.

## Оценки не являются вероятностями

Оценки поиска и SET — сигналы ранжирования. Текстовые оценки CLAP, оценки MERT по референсному треку,
SET, Hybrid и пороги Audio Dedup нельзя считать одной шкалой.

## SET требует полного покрытия

Smart Set Builder требует SONARA и эмбеддинги MERT, MAEST и CLAP. Частично проанализированную
фонотеку можно просматривать и искать в других вкладках, но число подходящих для SET треков будет
небольшим.

## Готовность SONARA требует точных активных контрактов

Сводка фонотеки считает строку SONARA Core, только если она соответствует точному активному контракту
`core` и релизу. Покрытие отдельного трека независимо проверяет `timeline`, `embedding` и
`fingerprint` по их точным активным контрактам и актуальной идентичности трека. Активный релиз — это
один подготовленный набор из четырёх контрактов; отсутствующий или устаревший выход не считается
готовым.

Поскольку v7 — схема для создания новой базы, а не цель миграции на месте, процесс подготавливает
и активирует четыре контракта загруженного релиза до повторного анализа чистого состояния SONARA.
Этот полный набор проекта не запрашивает теги SONARA или её жанровую модель. См.
[контракт SONARA v0.2.9](../reference/sonara-v0-2-9-contract.md).

## Предпросмотр в браузере зависит от существования файлов

База хранит пути. Если файл перемещён или исчез, предпросмотр не заработает до повторного
сканирования или переноса сохранённых путей.

## Системные диалоги зависят от локальной графической среды

Выбор базы и папки использует локальный GUI. Если он недоступен, введите путь вручную или
используйте CLI.

## Документация собирается отдельно

Бэкенд раздаёт `/docs/` только при наличии `docs/dj-track-similarity/site/`. Для сборки запустите
`npm run check` из папки документации.
