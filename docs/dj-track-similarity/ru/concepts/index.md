# Основные понятия

> Для кого: Для тех, кто хочет понимать данные в интерфейсе.
> Задача: Объяснить основные типы данных и шкалы оценок без погружения в разработку.
> Тип: Оглавление понятий

Приложение сочетает файловые метаданные, аудиопризнаки, эмбеддинги и необязательные оценки локальных
классификаторов. Каждый источник отвечает на свой вопрос. Считайте их локальными сигналами
ранжирования и фильтрации, а не общей мерой музыкальной истины.

Используйте оценки, чтобы выбрать треки для прослушивания. Решение о месте в сете принимайте на слух.
