# Известные ограничения

> Для кого: Для тех, кто проверяет, ожидаемо ли текущее поведение.
> Задача: Перечислить границы, не выдавая их за ошибки.
> Тип: Помощь

## Локальная работа зависит от локальной среды

Приложению нужны ваша среда Python, FFmpeg, необязательные ML-пакеты, среда GPU, пути файловой
системы и доступ браузера к бэкенду.

## Анализ моделей может быть тяжёлым

MAEST, MERT, MuQ и CLAP требуют дополнительных ML-зависимостей. Большая библиотека занимает время и
память. MuQ работает с аудио `float32` 24 кГц и сейчас используется только в LAB Reference Compare,
не в SET, Hybrid, Audio Dedup или классификаторах. Сначала задайте малый `--limit`, затем настройте
устройство и партии.

## Оценки не являются вероятностями

Оценки поиска и SET — сигналы ранжирования. Текстовые оценки CLAP, оценки MERT по опорному треку,
SET, Hybrid и пороги Audio Dedup нельзя считать одной шкалой.

## SET требует полного покрытия

Smart Set Builder требует SONARA и эмбеддинги MERT, MAEST и CLAP. Частично проанализированную
библиотеку можно просматривать и искать в других вкладках, но число подходящих для SET треков будет
небольшим.

## Актуальное покрытие SONARA шире одного профиля

Сводка библиотеки считает любой подписанный профиль SONARA с актуальными версией, схемой, режимом,
частотой, диапазоном BPM и ревизией проекта. Это не доказывает, что каждая строка имеет точно полный
профиль. При проверке миграции смотрите задачи анализа и примеры сигнатур. Полный профиль проекта
означает все поддерживаемые адаптером аудиосемейства, но не запрашивает теги SONARA или её жанровую
модель. См. [контракт SONARA v0.2.9](../reference/sonara-v0-2-9-contract.md).

## Прослушивание зависит от существования файлов

База хранит пути. Если файл перемещён или исчез, прослушивание не заработает до повторного
сканирования или переноса сохранённых путей.

## Системные диалоги зависят от локальной графической среды

Выбор базы и папки использует локальный GUI. Если он недоступен, введите путь вручную или
используйте CLI.

## Документация собирается отдельно

Бэкенд раздаёт `/docs/` только при наличии `docs/dj-track-similarity/site/`. Для сборки запустите
`npm run check` из папки документации.
