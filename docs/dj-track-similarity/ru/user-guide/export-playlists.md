# Экспортируйте черновик плейлиста

> Для кого: Для тех, кто собрал временный сет и хочет получить файл вне приложения.
> Задача: Экспортировать M3U или CSV и понимать, что именно записывается.
> Тип: Руководство

::: warning Состояние интерфейса v7
Описанный ниже сценарий React относится к отложенному интерфейсу. Он ещё не перенесён на API схемы
v7, поэтому эти шаги сейчас не проверены и недоступны для v7. Используйте приведённую ниже
альтернативу через API бэкенда.
:::

## Актуальная альтернатива v7

Вызовите `POST /api/export` и передайте `name`, `track_ids`, `output_dir` и `format`. Формат должен
быть `m3u` или `csv`; ответ вернёт созданный `path`. Укажите ID треков явно, потому что актуальный
бэкенд v7 не хранит временное состояние текущего сета из отложенного браузерного интерфейса. См.
[справочник API](../reference/api.md).

## Сценарий отложенного интерфейса

Текущий сет остаётся временным до экспорта. Результаты поиска и черновики SET изменяют его только
после явного добавления треков.

## Действия в интерфейсе

1. Добавьте треки в текущий сет из фонотеки, результатов поиска, черновика SET или другой области интерфейса.
2. Укажите название плейлиста.
3. Выберите или введите выходную папку.
4. Нажмите кнопку экспорта M3U или CSV.

Если выходной папки нет, приложение создаст её.

## Имена файлов

Экспортёр преобразует название плейлиста в безопасное имя: сохраняет буквы, цифры, `.`, `_` и `-`,
заменяет остальные последовательности символов на `_`, а если после очистки имя пусто — использует
`playlist`.

## Формат M3U

Файл M3U содержит:

- строку `#EXTM3U`;
- по одной строке `#EXTINF:-1,Artist - Title` для каждого трека с исполнителем и названием;
- сохранённый локальный путь к каждому треку.

## Формат CSV

Файл CSV содержит столбцы:

```text
artist,title,bpm,key,energy,path
```

## Сохранение в коллекцию Rhythm Lab

Текущий сет также можно сохранить как коллекцию Rhythm Lab. Эта операция записывает данные в базу меток
Rhythm Lab, а не исходные аудиофайлы.

## Приватность

Экспорт содержит локальные пути и может раскрывать структуру коллекции. Считайте M3U и CSV личными
файлами, если вы намеренно не удалили из них приватные сведения.
