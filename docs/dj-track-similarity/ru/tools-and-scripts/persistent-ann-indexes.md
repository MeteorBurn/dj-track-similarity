# Постоянные индексы ANN

> Для кого: Для тех, у кого большая проанализированная библиотека и частый векторный поиск.
> Задача: Создать, проверить, измерить, использовать и очистить необязательные вспомогательные индексы.
> Тип: Руководство

Постоянные индексы ANN — создаваемые вспомогательные файлы для сохранённых эмбеддингов. Они
необязательны: точный поиск продолжает работать без них.

## Что хранится

По умолчанию индексы находятся рядом с выбранной базой SQLite:

```text
.dj-track-similarity-indexes/
```

Они содержат созданные артефакты векторного индекса и манифесты по одному адаптеру: `mert`, `maest`
или `clap`. Аудиофайлы не копируются, новые строки SQLite не создаются.

## Установите необязательный бэкенд

Дополнительный набор `ann` устанавливает `hnswlib`:

```powershell
python -m pip install -e ".[ann,dev]"
```

CLI поддерживает `auto`, `hnswlib` и `exact-numpy`. Режим `auto` предпочитает hnswlib, если он
доступен.

## Создайте и проверьте

```powershell
dj-sim index build --adapter clap --db .\data\library.sqlite
```

```powershell
dj-sim index verify --adapter clap --db .\data\library.sqlite
```

Адаптер выбирается через `--adapter` или `--embedding-key`.

Необязательные параметры сборки:

```powershell
dj-sim index build --adapter clap --backend auto --ef-construction 200 --m 16 --ef-search 100 --db .\data\library.sqlite
```

## Измерьте полноту

```powershell
dj-sim index benchmark --adapter clap --recall-k 50 --seed-count 20 --db .\data\library.sqlite
```

Бенчмарк сравнивает индекс с точным поиском и сообщает результат выбранного порога.

## Используйте в текстовом поиске CLAP

В CLI индекс включается явно:

```powershell
dj-sim text-search "warm dub techno pads" --use-ann-index --db .\data\library.sqlite
```

Если вспомогательный файл отсутствует, устарел или не поддерживается, команда предупреждает и
возвращается к точному поиску.

## Пересобирайте после изменения эмбеддингов

Пересоберите индекс после анализа соответствующего адаптера, сброса эмбеддингов, замены базы или
перемещения каталога индекса.

## Очистите созданные файлы

Один адаптер:

```powershell
dj-sim index clear --adapter clap --db .\data\library.sqlite
```

Все вспомогательные индексы в разрешённом каталоге:

```powershell
dj-sim index clear --db .\data\library.sqlite
```

Эти индексы — создаваемое локальное состояние, которое не следует добавлять в Git.
