# Find compatible tracks around a reference

> Audience: DJs looking for a next-track candidate.
> Goal: Use seed search and transition clues without over-trusting scores.
> Type: how-to

## Steps

- Add the reference track as a seed.
- Run SONARA for feature neighbors or MERT for embedding neighbors.
- Compare BPM, key relation, energy, and score breakdowns.
- Preview candidates in context before adding them to a set.

## Tempo note

Half/double tempo matching helps transition compatibility. It does not change the actual BPM trajectory used by explicit SET BPM modes.
