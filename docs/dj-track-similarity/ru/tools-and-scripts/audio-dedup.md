# Audio Dedup

> Audience: Пользователи этой страницы.
> Type: how-to

Default mode writes JSON/XLSX/log reports and deletes nothing: `.\.venv\Scripts\python.exe tools\audio-dedup\audio_dedup_cli.py --db <library-db> --root <music-folder>`. `--apply` is destructive and requires exact confirmation `APPLY DELETE`.
